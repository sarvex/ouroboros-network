Description

This module provides simulation environment and a snocket implementation suitable for IOSim.

Though this module is designed for simulation / testing, it lives in the library, since it is needed in `ouroboros-network-framework:test` and `ouroboros-network:test' components.

data BearerInfo Source #

Each bearer info describes outbound and inbound side of a point to point bearer.

Constructors

BearerInfo 

Fields

  • biConnectionDelay ∷ !DiffTime

    How long it take to create a connection

  • biInboundAttenuation ∷ Time → Size → (DiffTime, SuccessOrFailure)

    attenuation of inbound side of the bearer, i.e. attenuation used by bearers that were accepted.

  • biOutboundAttenuation ∷ Time → Size → (DiffTime, SuccessOrFailure)

    attenuation of outbound side of the bearer, i.e. the attenuation used by bearers that were created with connect call.

  • biInboundWriteFailure ∷ !(Maybe Int)

    Maximum number of successful writes for an inbound bearer.

  • biOutboundWriteFailure ∷ !(Maybe Int)

    Maximum number of successful writes for an outbound bearer.

  • biAcceptFailures ∷ !(Maybe (DiffTime, IOErrType))

    Time after which accept will throw an exception.

    Currently it only supports two kinds of exceptions, ones that are caught and rethrown by the server (ECONNABORTED), and an exception which would be caught, and delivered to the application via AcceptFailure.

  • biSDUSize ∷ !SDUSize

    SDU size of the bearer; it will be shared between outbound and inbound sides.

    Note: shrinking SDUSize means make it larger, as this allows to send fewer chunks through the bearer.
